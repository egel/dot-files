# Dot-files
Dot-files for Ubuntu 14.04

Contains:

  - [`Git`]()
    - configuration
    - useful aliases
  - [`Vim`]()
    - configuration
    - general most useful modules)
  - [`conky`]()
    - configuration
    - themes
  - bash aliases


## Installation

    $ cd ~/ && git clone --recursive git@github.com:egel/dot-files.git

## Update

    $ cd ~/dot-files/ && git submodule update --init --recursive
